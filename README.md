Здесь собраны проекты, демонстрирующие мои навыки в разных областях Data Science:

📊 Анализ данных, статистика

- Секреты темнолесья - исследовательский анализ топа игровых платформ, а также деление игр по категориям
- Секреты темнолесья sql - изучение влияния характеристик игроков и их игровых персонажей на покупку внутриигровой валюты, а также оценнка активности игроков при совершении внутриигровых покупок
- Яндекс афиша - анализ востребованности мероприятий, влияние на средний чек и пользовательскую активность
- Проверка гипотезы - в ч. 1 проведение А/В тестирования, в ч. 2 оценка результатов проведенного А/В теста
- Кафе и рестораны - анализ типа, месстоположения и рейтинга заведений для открытия новой точки
- Стартапы - исследовательский анализ ключевых метрик стартапов для инвестирования средств
- Разработка А/В тестирования и анализ результатов - проверка гипотезы с помощью А/В тестирования
- Агенство недвижимости - анализ рынка недвижимости Санкт-Петербурга и Ленобласти
